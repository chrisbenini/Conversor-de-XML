## `> overview`

Este projeto foi criado para trabalhar com **estruturação, leitura e organização de dados em XML**, aplicando conceitos de **tratamento de dados**, **padronização estrutural** e **integração com Excel**.

Mesmo sendo um projeto acadêmico, ele foi desenvolvido com uma organização mais próxima de um cenário real, servindo também como material de estudo e portfólio.
